## Модули

### Модуль импорта данных (MVP - первая задача Development Agent)

**Файл**: `data-import.md`

**Назначение**: Импорт существующих данных из WhatsApp переписки в CRM систему.

**Статус**: MVP - первая задача Development Agent

**Описание**: Модуль для извлечения и импорта данных из ~100 переписок WhatsApp, включая клиентов, сообщения, определение статусов.

**Особенности**:
- Для MVP реализуется только импорт из WhatsApp
- Архитектурно заложено расширение на другие источники (Telegram, CSV, Email)
- Модуль спроектирован с учетом будущего расширения (абстракция `DataExtractor`)
