## Реализация

### Структура файлов

```
packages/database/
├── prisma/
│   ├── schema.prisma
│   └── seed.ts              # Seed скрипт
└── seeds/
    ├── organizations.ts     # Seed данные организаций
    ├── users.ts             # Seed данные пользователей
    ├── clients.ts           # Seed данные клиентов
    ├── products.ts          # Seed данные продуктов
    ├── tours.ts             # Seed данные туров
    ├── partners.ts          # Seed данные партнеров
    └── messages.ts          # Seed данные сообщений
```

### Команды

```bash
# Запуск seed данных
npm run db:seed

# Очистка и повторный seed
npm run db:reset

# Seed только для разработки
npm run db:seed:dev
```

### Примечания

1. **Реальные данные**: После предоставления пользователем реальных данных из WhatsApp, seed данные будут обновлены
2. **Безопасность**: Пароли в seed данных только для dev окружения
3. **Изоляция**: Seed данные создаются для одной организации (Soul KG)
4. **Версионирование**: Seed данные версионируются вместе с миграциями
